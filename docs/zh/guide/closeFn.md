# 关闭组件回调函数

关闭组件回调函数用到的配置如下

| 字段        | 类型    | 默认值 | 备注                         |
| ----------- | ------- | ------ | ---------------------------- |
| closeFn     | Function | null  | 关闭组件回调函数               |

## 回调参数

回调函数会返回以下参数

| 参数        | 类型    | 默认值 | 备注                         |
| ----------- | ------- | ------ | ---------------------------- |
| way     | string | null  | 关闭方式，详细见下表               |

way 说明

| 参数        | 说明                         | 备注                        |
| ----------- | ---------------------------- |---------
| esc     | 按下 esc 键关闭               | 需开启键盘控制支持               |
| mask     | 点击遮罩层关闭               | 需开启点击遮罩层关闭支持               |
| closeBtn     | 点击图片关闭               |

## 示例

```js
this.$hevueImgPreview({
  // ...
  closeFn: (data) => {
    // data => { way: 'esc' }
    // 用户通过键盘 esc 关闭组件
  }
})
```
